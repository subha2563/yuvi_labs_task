# PxP Array Generator and Analyzer

## Objective
This program generates a PxP table containing random even numbers between 2 and 16 (inclusive) and performs a selective analysis based on user input.

## Features
*   Dynamically creates a 2D array based on a user-provided size `P`.
*   Displays the data in a neatly formatted terminal grid.
*   Highlights a specific even number (2-16) chosen by the user and redraws the grid.
*   Calculates and prints the total occurrences of the highlighted number.
*   Safely handles edge cases such as non-integer inputs (e.g., "foo") and out-of-range numerical entries.

## How to Run
1. Ensure you have the Java Development Kit (JDK) installed on your system.
2. Open a terminal and navigate to the directory containing this repository.
3. Compile the program using the command: 
   `javac Table.java`
4. Run the compiled program using the command: 
   `java Table`
5. Follow the on-screen prompts to input the array size and the number to highlight.
